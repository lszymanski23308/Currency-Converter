# Opis 
- Kalkulator który powstał jako jedna z pierwszych prac w kursie **YouCode.**
- Kalkulator pozwala na przeliczenie ze złotówek 3 różnych walut,takich jak:
1. euro 
1. dolar amerykański 
1. funt szterling
- W kodzie użyto konwencji BEM.
